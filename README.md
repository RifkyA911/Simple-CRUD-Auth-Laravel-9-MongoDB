<h1 align="center">Simple CRUD Auth — Laravel 9 + MongoDB</h1>
<p align="center"><em>Authentication and CRUD application with Laravel 9 and MongoDB</em></p>

---

## English

A web application featuring user authentication and full CRUD operations built with Laravel 9 and MongoDB.

### Features
- User registration and login
- Full CRUD operations
- Authentication middleware
- Form validation
- Simple Blade templates

### Requirements
- PHP 8.0+
- Composer
- MongoDB
- Laravel 9

### Installation
```
git clone https://github.com/RifkyA911/Simple-CRUD-Auth-Laravel-9-MongoDB.git
cd Simple-CRUD-Auth-Laravel-9-MongoDB
composer install
cp .env.example .env
php artisan key:generate
php artisan serve
```

### Tech Stack
- Backend: Laravel 9
- Database: MongoDB (jenssegers/mongodb)
- Frontend: Blade + Bootstrap
